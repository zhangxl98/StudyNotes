网络配置基本命令

  1. 查看本地网卡

    ifconfig -a

  2. 激活网卡

    ifconfig eth0 up

  3. 重启网络服务

    /etc/init.d/networking restart

  4. DHCP 网络自动获取 IP

    dhclient eth0

  5. 手动设置 IP

    ifconfig eth0 [inet] netmask [netmask]

  6. 设置网关

    route add default gw [gateway]

  7. 静态 IP 配置文件

    iface eth0 inet static
address [inet]
netmask [netmask]
gateway [gateway]

  8. DNS 配置文件

    /etc/resolv.conf
